You can do some of the following things: # Get the URL of a successful deployment gif GET /success # Get the URL of a failed deployment gif GET /fail # Redirect to the URL of a successful deployment gif GET /success?redirect=1 # Redirect to the URL of a failed deployment gif GET /fail?redirect=1 # Get the URL of a successful deployment gif in json GET /success?json=1 or GET /success.json # Get the URL of a failed deployment gif in json GET /fail?json=1 or GET /fail.json
